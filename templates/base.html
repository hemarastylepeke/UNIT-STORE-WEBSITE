ABOUT

The website was designed by MATHEMATICS AND COMPUTER SCIENCE STUDENTS at Kenyatta University as an HCI Project. The project acts as an Online Unit Store where students can upload notes given by their teachers in class and any other academic document in pdf, powerpoint and word. This will help teachers evade Notes duplication as all these notes will be stored in a single web page. Papers can also be scanned in pdf format and then uploaded to the server for storage. Alongside are some recommended sites for Extra books.
